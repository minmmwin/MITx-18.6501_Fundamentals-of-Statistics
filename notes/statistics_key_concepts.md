# 🧠 Statistics — Key Concepts (MITx 18.6501)

This document summarizes key statistical concepts learned in the course.  
Focus is on intuition and understanding rather than formal proofs.

---

## 📌 1. Descriptive Statistics

Summarize data using:
- Mean (average)  
- Median (middle value)  
- Variance / standard deviation (spread)  

👉 Gives a quick understanding of data distribution.

---

## 📌 2. Sampling

Data is often collected as a sample from a population.

👉 Goal: use sample → make conclusions about population.

---

## 📌 3. Estimation

Estimate unknown parameters (e.g., mean μ).

Types:
- Point estimate (single value)  
- Interval estimate (range of values)  

👉 Example: sample mean estimates population mean.

---

## 📌 4. Bias & Variance

- Bias: how far estimate is from true value  
- Variance: how much estimate varies across samples  

👉 Trade-off between accuracy and stability.

---

## 📌 5. Confidence Intervals

Range that likely contains true parameter:

Example: 95% confidence interval  

👉 “If we repeat many times, 95% of intervals contain the true value.”

---

## 📌 6. Hypothesis Testing

Used to make decisions from data.

Steps:
1. Define null hypothesis (H₀)  
2. Define alternative hypothesis (H₁)  
3. Compute test statistic  
4. Use p-value to decide  

👉 Helps determine if an effect is statistically significant.

---

## 📌 7. p-value

Probability of observing data (or more extreme) under H₀.

👉 Small p-value → evidence against H₀.

---

## 📌 8. Type I & Type II Errors

- Type I: reject true H₀ (false positive)  
- Type II: fail to reject false H₀ (false negative)  

👉 Trade-off controlled by significance level (α).

---

## 📌 9. Maximum Likelihood Estimation (MLE)

Choose parameter that maximizes likelihood of observed data.

👉 Finds best-fit model based on data.

---

## 📌 10. Bayesian vs Frequentist

- Frequentist: parameters are fixed, data is random  
- Bayesian: parameters are random, updated with data  

👉 Bayesian uses prior + data → posterior.

---

## 📌 11. Regression (Basics)

Model relationship between variables:

Example:
y = β₀ + β₁x + error  

👉 Used for prediction and understanding relationships.

---

## 🎯 Key Takeaway

Statistics provides tools for:
- making decisions under uncertainty  
- estimating unknown quantities  
- testing hypotheses using data
