# MITx 18.6501 — Fundamentals of Statistics

This repository documents my learning progress for **MITx 18.6501**, part of the MITx MicroMasters Program in Statistics and Data Science.

The course focuses on building a strong foundation in **statistical inference**, including estimation, hypothesis testing, and data-driven decision making.

---

## 📚  Course Overview

- Institution: Massachusetts Institute of Technology (MITx)  
- Platform: edX  
- Status: In Progress  
- Focus: Statistical inference, estimation, and hypothesis testing  

---

## 🧠 Topics Covered

- Descriptive statistics  
- Sampling and data distributions  
- Point estimation and properties  
- Confidence intervals  
- Hypothesis testing  
- Likelihood and maximum likelihood estimation (MLE)  
- Bayesian vs frequentist perspectives  
- Regression fundamentals  

---

## 🎯 Skills Developed

- Statistical reasoning and inference  
- Interpreting data and uncertainty  
- Designing and evaluating hypotheses  
- Understanding estimation methods  
- Applying statistical concepts to real-world problems
